ScitoBot
========

Building and Deploying
----------------------
```
$ git clone git://github.com/patfair/ucpp.git
$ less ucpp/README # Set up ucpp for your env. 
...
$ git clone git://github.com/leroux/scitobot.git
$ cd scitobot
$ ucpp init && ucpp configure
$ make
$ make deploy
```
